# 🧮 Number Challenge

A modern desktop math game built with **Python** and **CustomTkinter**.

## ✨ Features

- 🎲 Random math questions
- ➕ ➖ ✖ ➗ Four operations
- 🏆 High Score system
- ⚡ Best Time record
- ⏱ Countdown timer
- 💡 Hint button
- 🔊 Sound effects
- 🌙 Dark / Light mode
